📸 РУКОВОДСТВО ПО ПОДГОТОВКЕ ИЗОБРАЖЕНИЙ
Требования к фотографиям:
1. Главное фото (Hero Section):
Размер: 1200x1600px (вертикальное)
Формат: JPG или WebP
Качество: Высокое (80-90%)
Совет: Выбирайте фото, где вы смотрите друг на друга или целуетесь
Имя файла: couple-main.jpg
2. Фото для галереи:
Размер: 800x1000px (вертикальные)
Количество: 3-6 фото
Формат: JPG или WebP
Имена файлов: photo1.jpg, photo2.jpg, photo3.jpg
Совет: Выберите фото из разных периодов отношений
3. Фото места проведения:
Размер: 1200x800px (горизонтальное)
Формат: JPG
Использование: Можно добавить в секцию Location
Оптимизация изображений:
Перед загрузкой обязательно оптимизируйте фото:
Онлайн-инструменты:
TinyPNG - сжатие без потери качества
Squoosh - конвертация в WebP
CompressJPEG - пакетное сжатие
Рекомендации:
Максимальный размер файла: 300KB для каждого фото
Используйте формат WebP для лучшей производительности
Создайте fallback в JPG для старых браузеров
Цветовая палитра (из вашего скриншота):
Основные цвета:
Бордовый: #8B1E3F (основной акцент)
Пыльная роза: #C9A9A6 (вторичный)
Слоновая кость: #F5E6E8 (фон)
Темно-синий: #2C3E50 (текст)
🗺️ ИНТЕГРАЦИЯ YANDEX MAPS
Шаг 1: Создание метки
Перейдите на Yandex Map Constructor
Нажмите "Создать карту"
Найдите адрес вашего ресторана
Поставьте метку
Настройте внешний вид метки (цвет: бордовый #8B1E3F)
Сохраните карту
Шаг 2: Получение кода
Нажмите "Получить код"
Скопируйте координаты (широта и долгота)
Вставьте их в js/main.js вместо [59.9343, 30.3351]
Альтернатива: Простая вставка iframe
Если конструктор кажется сложным, можно просто вставить iframe:
html
1234567
<iframe 
    src="https://yandex.ru/map-widget/v1/?ll=30.3351%2C59.9343&z=15&pt=30.3351,59.9343" 
    width="100%" 
    height="450" 
    frameborder="0" 
    allowfullscreen="true">
</iframe>
📝 YANDEX FORMS (RSVP)
Создание формы:
Перейдите на Yandex Forms
Создайте новую форму
Добавьте поля:
Имя и фамилия (текстовое поле, обязательно)
Сможете ли вы прийти? (выбор из вариантов: Да/Нет, обязательно)
Количество гостей (число, если "Да")
Предпочтения по еде (множественный выбор)
Пожелания и комментарии (текстовая область)
Контактный телефон (телефон, обязательно)
Email (email)
Настройте дизайн (цвета: #8B1E3F)
Сохраните форму
Нажмите "Встроить" → скопируйте ID формы
Замените YOUR_FORM_ID в index.html на ваш ID
